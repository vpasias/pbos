# VLAN case
Hypervisor Hosts Interfaces:
- eth0 - admin (PXE-admin)
- eth1 - cluster (mgmt-overlay-rados GW)
- eth2 - storage
- eth3 - external (provider)

interfaces & VLANs allocation:
- eth0: admin network: 192.168.254.0/24
- eth1 (VLAN 0 - native): mgmt network: 192.168.200.0/24
- eth1.10 (VLAN 10): overlay network: 192.168.210.0/24
- eth1.20 (VLAN 20): rados GW network: 192.168.220.0/24 
- eth2 (VLAN 0 - native): provider network: 192.168.230.0/24 (in real implemenation: eth1.30 - VLAN 30)
- eth3 (VLAN 0 - native): storage network: 192.168.250.0/24
